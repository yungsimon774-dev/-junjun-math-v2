上传方法：
1. 解压 ZIP。
2. 在原来的 GitHub 仓库选择 Add file / Upload files。
3. 把以下文件全部上传并覆盖：
   index.html
   app.js
   style.css
   config.js
   courses.js
   questions.js
   click.wav
   correct.wav
   wrong.wav
   finish.wav
4. 提交后等待 1 至 3 分钟。
5. 刷新网站；如仍显示旧版，在网址最后加 ?v=30 再打开。

不要只上传 ZIP。
