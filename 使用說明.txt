峻峻數學大冒險 V2.1 iPhone平鋪外接有聲版

這版專門適合 iPhone + Safari + GitHub 網頁上傳。

倉庫首頁必須直接看到：
index.html
app.js
style.css
config.js
courses.js
questions.js
click.wav
correct.wav
wrong.wav
finish.wav

更新方式：
1. 解壓縮 ZIP。
2. 在原來 GitHub 倉庫選 Add file / Upload files。
3. 把以上全部檔案重新上傳。
4. GitHub 顯示同名檔案時，直接提交覆蓋。
5. 不用建立新倉庫，不用建立 data 或 audio 資料夾。
6. 上傳完成後等 1 至 3 分鐘，再重新整理網站。
7. 如果仍顯示舊畫面，可在網址最後加 ?v=21 再開一次。

注意：
- 不要只上傳 ZIP。
- 不要漏掉 config.js、courses.js、questions.js。
- 不要漏掉四個 WAV 音效。
- 第一次使用聲音，要先點一下頁面按鈕。
